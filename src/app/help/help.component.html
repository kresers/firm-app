Bienvenue sur OpenAnnuaire

OpenAnnuaire récupère depuis une base de données libre d'utilisation, les informations sur toutes les entreprises françaises.
Un tableau, une map et des exports vous sont proposés afin de récupérer les données des entreprises qui vous intéresse.
De nombreux filtres sont à votre disposition pour affiner vos recherches.

Voici comment filtrer les entreprises sur OpenAnnuaire :

  1. Les + et - permettent d'afficher/cacher les champs à remplir
  2. Pour valider votre saisie, appuyer sur la touche entrée ou cliquez n'importe où en dehors du champ
  3. Répétez l'opération sur les filtres souhaités, à chaque fois, en haut de page, une barre de progression s'affiche pour vous tenir informer de l'avancement de votre requête.
  4. Pour retirer un filtre en particulier : cliquez sur x à coté du filtre à supprimer.
  5. Un filtre général, représenté par dans la barre supérieure du site vous permet de rechercher n'importe quelle information liée à une entreprise.
  6. Le tableau, la map ainsi que les exports sont automatiquement mis à jour avec les données souhaitées.
